## **敘述**
以前使用其他人寫的套件都要下載，自己在放到專案內好讓程式引用套件，通常不會只引用一二個套件問題就來了。

有些套件可能會重複名稱，有些可能規定要放跟目錄，有些有相依性信套件要一併下載，相依性套件可能重複。

這讓引用套件有時變得很混亂不好控管，所以後來就有人學習了套件相依管理系統誕生了`Cmposer`。

現在很多PHP軟體已經都透過composer來管理專案了，他可以有效的控制專案使用的相依套件版本‧
