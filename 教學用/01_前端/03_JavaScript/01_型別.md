# 教學用 > 前端 > JavaScript > 型別

## **Description [描述]**
JavaScript是種強行別程式語言，所以需要對型別有一定的認知，型別間是無法直接做相容的，必須透過一些手段來達成。

## **Teaching & Examples [教學&範例]**
### 基本型別 (Primitives)
1. String
2. Number
3. Boolean
4. Null
5. Undefined
6. Symbol

### 物件型別 (Object)
1. Array - 陣列
2. Object - 物件
3. Function - 函式

### 查詢型別
```js
let data = {};
console.log(typeof data);
// Object
```
