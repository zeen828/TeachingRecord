## **Teaching & Examples [教學&範例]**
### 安裝(Laravel預設是沒安裝的需要自行加裝)
```bash
composer require predis/predis
```

### 建立
```php
use Illuminate\Support\Facades\Redis;

Redis::set('rest:key', 'Test Data');

// 時效性(key, 秒數, 資料)
Redis::setex('rest:key', 300, 'Test Data');
```

### 取用
```php
use Illuminate\Support\Facades\Redis;

Redis::get('rest:key')
```

### 檢測是否存在
```php
use Illuminate\Support\Facades\Redis;

Redis::exists('rest:key')
```
