## **敘述**

## **教學&範例**
### 呼叫函式
```go
package main
import (
    "fmt"
)

// 自訂函式
func hello(name string) {
    fmt.Println(fmt.Sprintf("Hello %s", name))
}

// 主程式的開端
func main() {
    // 使用自訂函式
    hello("小貓")
    hello("小狗")
    hello("小豬")
}
```
