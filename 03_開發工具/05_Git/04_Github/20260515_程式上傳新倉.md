### 進入專案

### 設定作者資訊

### 設定 GitHub 遠端位址
git remote add origin https://github.com/zeen828/Demo_ChianIn.git

### 將分支名稱改為 main
git branch -M main

### 授權問題
GitHub 為了安全考量，現在不接受使用「帳號密碼」來執行 Git 指令。
你看到的錯誤訊息 Password authentication is not supported 就是在要求你使用 Personal Access Token (PAT)。

1. 產生 GitHub Token (PAT)
	1. 登入 GitHub 網頁。

	2. 點擊右上角個人頭像 -> Settings。

	3. 左側選單拉到最底，點擊 Developer settings。

	4. 點擊 Personal access tokens -> Tokens (classic)。

	5. 點擊 Generate new token (選 classic 版)。

	6. Note 隨便填（例如：MyLaravelDev），Expiration 建議選 90 天或更長。

	7. Select scopes：務必勾選第一項 repo (全部子項目)。

	8. 滑到最下方點擊 Generate token。

	9. 重要： 複製那一串以 ghp_ 開頭的字串，把它存起來，因為離開頁面後就再也看不到了。


### 推送到 GitHub 並設定追蹤
git push -u origin main

	1. Username: 輸入 zeen828

	2. Password: 貼上你剛才複製的那串 ghp_ 開頭的 Token（輸入時畫面上不會顯示字元是正常的，直接按 Enter 即可）。
